## Описание
**TrustPrime 2.0** — это мощный и расширяемый инструмент для восстановления системных функций Windows, снятия программных блокировок и контроля за состоянием ОС. Программа разработана как универсальный помощник для пользователей, столкнувшихся с ограничениями, последствиями вирусов или нестабильной работой системы.

В отличие от традиционных «чистильщиков», TrustPrime фокусируется на **быстром возврате контроля** над системой: разблокировка реестра, мониторинг ресурсов в реальном времени и удобное управление процессами в одном интерфейсе.

## Основные возможности
- 🔓 **Разблокировка системных компонентов** — восстановление ключевых параметров Windows и возврат доступа к заблокированным функциям
- 📊 **Мониторинг в реальном времени** — нагрузка на CPU, RAM, диски и краткая сводка о системе
- ⚙️ **Управление процессами** — просмотр PID, потребления ресурсов, уровня критичности, завершение нагруженных задач и перезапуск Explorer
- 📦 **Единый исполняемый файл** — не требует установки, зависимостей или настройки. Работает сразу после запуска
- 🔄 **Модульная архитектура** — проект построен так, чтобы легко расширяться новыми функциями в будущих сборках

## Технические детали
-  **Язык разработки:** Python
-  **Сборка:** Скомпилировано в нативный `.exe` (однофайловая автономная версия)
-  **Права доступа:** Требуется запуск от имени администратора
-  **Совместимость:** Windows 10 / 11 (64-bit)
-  **Размер:** ~81.6 МБ

##  Как начать использовать
1. Скачайте архив `TrustPrime.exe` из раздела [Releases](https://github.com/0PusD0tNet/trustprime/releases)
2. Нажмите правой кнопкой на `TrustPrime.exe` → **Запуск от имени администратора**
3. Используйте вкладки интерфейса для мониторинга, разблокировки или управления процессами

>  При первом запуске Windows может показать SmartScreen: нажмите `Подробнее` → `Выполнить в любом случае`. Это стандартное поведение для неподписанных исполняемых файлов.
